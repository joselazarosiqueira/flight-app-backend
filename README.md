# flight-app-backend

This is a Spring Boot Application to manage simple Flights

## Requirements:
-PostgreSQL datasource: db_dev_atech database and sc_flight schema

## Run
mvn spring-boot:run -Dspring.profiles.active=dev

## Build

mvn clean install

## Running tests

mvn test
